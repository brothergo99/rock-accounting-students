核心 AI 工具介紹

🤖 Claude Code（Cowork）

AI 助理程式碼生成網站建置

原本是工程師用的 AI 輔助工具，現在多了助理功能，可以直接和它對話來建立完整產品（官網、資料整理、流程自動化）。
最厲害的地方：完全不用寫程式，全程用聊天完成。

🎙️ Whisper（語音轉文字）

免費繁體中文本機運行

開源語音辨識工具，可將錄影/錄音轉成逐字稿 .txt 檔。
模型等級：middle 模型可處理大多數繁體中文內容；需要一個重要的 bin 辨識檔才能運作。

🌐 GitHub Pages

免費托管正式網址無需伺服器

將 HTML 檔案上傳到 GitHub，即可獲得正式網址。
關鍵注意：首頁檔名必須命名為 index.html，否則網址無法正常顯示內容。

課後複習自動化流程
⚙️ Rock 的標準作業流程（SOP）
1
課後錄影下載
2
用 Whisper 跑逐字稿 → 產出 .txt 檔
3
將 .txt 丟入指定資料夾，告訴 Cowork「幫誰生成」
4
Cowork 自動產出一頁式 HTML，並上傳到 GitHub Pages
5
學生收到永久連結，隨時可回顧
個人官網 Landing Page 建置

🛠️ 從聊天到正式官網的步驟

1
告訴 Claude 你需要一個 Landing Page（課程介紹頁）
2
貼上你的自我介紹、課程亮點、社群連結（Threads、YouTube 等）
3
Claude 自動讀取並整合，生成 HTML 檔案
4
在 GitHub 申請帳號，建立 Repository，上傳 index.html
5
開啟 GitHub Pages → 取得正式網址，所有連結皆可點擊
6
接入 Google Analytics → 看即時流量與訪客來源
Google Analytics 流量分析

📊 看懂你的訪客來源

Direct：從社群點連結直接進來的人（你的粉絲）
Organic：從搜尋引擎或其他網站流入（被動曝光）
可看過去30分鐘即時人數，也可看每日趨勢
了解數據後才能判斷：要補內容、要宣傳、還是要提高回訪誘因
社群經營的 ⅓ 法則
❤️
⅓
喜歡你的人
專注服務他們，讓他們帶人來
😤
⅓
討厭你的人
不需要花力氣改變他們
😐
⅓
不認識你的人
靠內容慢慢拉近，做大基數

💡 流量節奏觀察

發訊息後，第一天大約 ⅓ 的人會看到；再過 3–4 天，第二批人陸續看到，累計可達 ⅔。
因此「本週數據低」不代表內容沒效果，要看一週完整趨勢，也要思考：給人回來的理由（定期更新、限時活動等）。

🎯 今日學習重點

Cowork 最強之處：不用寫程式，全程對話即可建出可上線的產品
Whisper：免費、本機，逐字稿轉出後讓 AI 讀，省下大量整理時間
GitHub Pages：免費正式網址，記得首頁命名 index.html
Google Analytics：接上後才能看流量、做決策

✅ 馬上可以嘗試的事

把 Whisper 安裝好，試跑一段錄音看逐字稿品質
用 Claude 描述自己的需求，看它能做出什麼 Landing Page
觀察自己社群的流量節奏，記錄第1天 vs 第5天的閱讀數差異
