💵 現金基礎 vs 應計（權責）基礎
💰 現金基礎
• 有收現金 → 認收入
• 有付現金 → 認費用
• 不管服務有沒有提供
📋 應計（權責）基礎
• 有事實存在 → 認收入
• 費用應發生 → 認費用
• 不管有沒有收/付現
現金基礎淨利 > 應計基礎淨利 的兩種情境：
情境① 有收現金，但尚未認收入
→ 現金基礎已認收入，應計尚未認
情境② 已認費用，但現金尚未支出
→ 應計基礎認費用，現金基礎未認 ✅ 本題答案
例：賒購文具用品 → 借費用/貸應付帳款，現金未出去
📌 重要分類提醒
利息費用 / 利息收入 營業外
文具用品費用（文具用品） 費用科目！不是資產
買辦公大樓 固定資產，非費用
⚠️ 「文具用品」聽起來像資產，考試最愛考這個陷阱！耗用時：借文具用品費用 / 貸用品盤存
📊 速動比率 vs 流動比率
流動比率 = 流動資產 ÷ 流動負債
速動比率 = （流動資產 − 存貨 − 預付） ÷ 流動負債
速動資產 = 能快速變現的資產
✅應收帳款

直接可用

❌預付費用
付掉了不會變現
❌用品盤存
拿來用的，不賣
❌固定資產
非流動
🧮 本題解析：速動比率計算
流動比率=3，流動負債=20萬，存貨占流動資產25%，預付費用=5萬
  • 流動資產 = 3 × 20萬 = 60萬
    流動比率 × 流動負債 = 流動資產
  • 存貨 = 60萬 × 25% = 15萬
    題目告知存貨占流動資產25%
  • 速動資產 = 60 − 15 − 5 = 40萬
    別忘記扣掉預付費用5萬！（最常漏扣）
  • 速動比率 = 40萬 ÷ 20萬 = 2
🎯 存貨增加不影響速動資產，但會讓流動負債增加 → 速動比率下降

📦 期末存貨影響口訣
銷貨成本 = 期初 + 本期進貨 − 期末存貨
期末存貨 高估 → 成本低估 → 淨利高估
期末存貨 低估 → 成本高估 → 淨利低估
📌 期末存貨與淨利同方向，與銷貨成本反方向
