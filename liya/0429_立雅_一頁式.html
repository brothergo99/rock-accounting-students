兩年抵消
  • 為何次期存貨無影響？前年期末存貨在次年全部賣出，一旦進入成本，期末存貨不再受影響。
  • 保留盈餘兩年抵消：本期高估、次期低估，一增一減最終歸零。
  • 113年考題考了毛利率＋周轉率：5個答案缺一不可，無蒙對捷徑。
🏦 銀行調節表
公司端 銀行端 各項目處理原則
📋 公司端調整
＋銀行已收、公司未記
（托收票據 ＋ 利息）
－銀行已扣、公司未記
（手續費 / 退票）
＋公司記錯少記
（支票多扣→加回差額）
🏛 銀行端調整
＋公司已存、銀行未入
（在途存款）
－公司已開、銀行未扣
（未兌現支票）
＋銀行誤扣本公司帳戶
→ 銀行端加回
▸ 含 X 題型 — 方程式解法（113年考法）
帳面餘額= X ， 銀行對帳單= 0.9X
X ＋（公司端各調整項）＝ 0.9X ＋（銀行端各調整項）
兩邊相等 → 解出 X → 代回求正確餘額
⚡ 易錯提醒
⚠ 方向混淆
期末存貨 ↑ → 銷貨成本 ↓（反向）；期末存貨 ↑ → 淨利 ↑（同向）。背口訣：「期末存貨與銷貨成本反，與淨利同。」
⚠ 銀行調節主體混淆
未兌現支票 → 銀行端減項。退票、手續費、托收 → 公司端處理。做題前先判斷主體，再定加減。
⚠ 計算毛利率、周轉率
毛利率＝（銷貨收入－調整後銷貨成本）÷ 銷貨收入。周轉率分母用調整後期初＋期末平均，兩端都要對。
📝 考場策略 — 存貨題
  • 先列三年度銷貨成本表與存貨表
  • 逐行逐年填入調整數，避免漏項
  • 確認銷貨成本、期初/末存貨全部算對，再求毛利率與周轉率
  • 15分題型分散在每個計算步驟，全對才能贏同場競爭者
📝 考場策略 — 銀行調節表
  • 遇含X題：先把所有項目分公司端/銀行端列完
  • 左右兩邊相等，列方程式解X
  • 時間不夠只算一邊也可得部分分數
  • 「銀行誤扣他帳戶」等奇葩情境=出題考試用的，正常處理即可
✏️ 課後自我檢測
  • X1期末存貨高估$10,000，X1/X2的成本、淨利、存貨、保留盈餘各為何？
  • 公司帳面$200,000，銀行$180,000，找出未兌現支票與在途存款差額。
  • 用自己的話解釋：為何前年高估存貨到次年就不影響存貨餘額？
